OpenAI API를 활용한 대화형 상품 추천 시스템

📋 프로젝트 개요

사용자와 자연어로 대화하며 상품을 추천하는 AI 기반 쇼핑몰 시스템입니다. OpenAI의 GPT-4와 임베딩 API를 활용하여 사용자의 의도를 파악하고, 벡터 검색을 통해 가장 적합한 상품을 추천합니다.

✨ 주요 기능

  • 대화형 상품 추천 - GPT-4를 활용한 자연스러운 대화
  • 벡터 유사도 검색 - 1536차원 임베딩 기반 상품 검색
  • 맥락 이해 - 대화 히스토리를 고려한 추천
  • 사용자 선호도 학습 - JSONB 기반 선호도 저장
  • 실시간 캐싱 - Redis를 통한 빠른 응답

핵심 컴포넌트

  • EmbeddingApiClient - OpenAI API 연동 및 임베딩 생성
  • ConversationalRecommendationService - 대화 관리 및 추천 orchestration
  • RecommendationEngine - 벡터 유사도 검색 엔진
  • ProductRepository - pgvector 기반 상품 검색

🚀 빠른 시작

1. 환경 설정

# application-secrets.properties 생성
openai.api.key=sk-proj-your-key-here
chatgpt.api.key=sk-proj-your-key-here

2. Docker 서비스 시작

docker-compose up -d

3. 애플리케이션 실행

./gradlew bootRun

4. API 테스트

curl -X POST http://localhost:8080/api/test/recommendation/text \
  -H "Content-Type: application/json" \
  -d '{"query": "파란색 셔츠 추천해줘"}'

📊 성능 최적화

  • IVFFlat 인덱스 - 벡터 검색 속도 10배 향상 (500ms → 50ms)
  • Redis 캐싱 - 동일 쿼리 재사용 시 즉시 응답
  • 배치 처리 - 여러 상품의 임베딩을 한 번에 생성
  • Connection Pooling - HikariCP 최적화 설정
